git add . : 수정한 파일 추가
git commit -m "커밋 메세지" : add한 파일 커밋
git pull origin master : 저장소에 있는 파일 받아오기
git push -u origin master : 커밋한 파일 저장소로 업로드

-------------------------------------
1. git add . - 수정한 파일 커밋 전 단계
2. git commit -m "전달할 텍스트입력 " - 수정한 파일 커밋
3. git pull - 원격 저장소의 커밋을 지역 저장소로 가져옴
4. git push - 업로드
------------------------------------
